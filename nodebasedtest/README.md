# Node-Based Editor

바닐라 JavaScript로 구현한 노드 기반 시각적 스크립팅 에디터입니다. Python Node Editor 프로젝트를 참고하여 웹 브라우저에서 동작하는 버전으로 개발되었습니다.

## 기능

- **노드 생성**: 다양한 타입의 노드를 생성할 수 있습니다
  - 숫자 노드: 숫자 값을 설정
  - 수학 노드: 덧셈, 뺄셈, 곱셈, 나눗셈 연산
  - 문자열 노드: 문자열 값을 설정
  - 출력 노드: 결과를 출력 패널에 표시

- **노드 조작**
  - 드래그 앤 드롭으로 노드 이동
  - 노드 삭제 (× 버튼 또는 Delete 키)
  - 노드 선택 및 다중 선택

- **노드 연결**
  - 출력 핀에서 입력 핀으로 연결
  - 베지어 커브를 사용한 시각적 연결선
  - 연결된 노드 간 데이터 플로우

- **그래프 실행**
  - 노드 네트워크를 실행하여 결과 계산
  - 출력 패널에 결과 표시

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다
2. 툴바에서 원하는 노드 타입을 클릭하여 노드를 추가합니다
3. 노드 헤더를 드래그하여 위치를 이동할 수 있습니다
4. 출력 핀(오른쪽)을 클릭하고 입력 핀(왼쪽)으로 드래그하여 연결합니다
5. "실행" 버튼을 클릭하여 그래프를 실행합니다
6. 결과는 하단 출력 패널에 표시됩니다

## 노드 타입

### 숫자 노드
- 숫자 값을 설정하고 출력합니다
- 입력 필드에서 값을 직접 수정할 수 있습니다

### 수학 노드
- 두 개의 입력(A, B)을 받아 연산을 수행합니다
- 연산 타입: 덧셈(+), 뺄셈(-), 곱셈(×), 나눗셈(÷)
- 입력 값이 연결되지 않은 경우 기본값(0)을 사용합니다

### 문자열 노드
- 문자열 값을 설정하고 출력합니다
- 입력 필드에서 텍스트를 직접 수정할 수 있습니다

### 출력 노드
- 입력된 값을 출력 패널에 표시합니다
- 여러 출력 노드를 사용하여 여러 값을 확인할 수 있습니다
